JavaScript模块开发标准

post.posted 2018-03-29

  • CommonJS

    服务端NodeJS模块规范, 通过require同步加载依赖, 通过exports或module.exports生成模块接口

  • AMD

    Asynchronous Module Definition,声明模块时指定所有的依赖, 异步加载, 依赖前置

  • CMD

    Common Module Definition, AMD的依赖延迟加载版

  • ES6模块

    引入关键字import, export, module. 在编译时就能确定模块的依赖关系.
